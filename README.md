# 💱 Conversor de Moedas em Java

![Java](https://img.shields.io/badge/Java-17%2B-orange)
![ExchangeRate-API](https://img.shields.io/badge/ExchangeRate--API-v6-blue)
![Gson](https://img.shields.io/badge/Gson-2.10.1-green)

Projeto de conversor de moedas desenvolvido em Java puro, consumindo a API pública ExchangeRate-API, taxas de câmbio em tempo real.

## 📋 Pré-requisitos

- **Java JDK 11+** (necessário para HttpClient nativo)
- **Biblioteca Gson** (para manipulação de JSON)
- **IDE** (IntelliJ, Eclipse, VS Code ou qualquer editor de texto)
- **Chave de API** gratuita do [ExchangeRate-API](https://www.exchangerate-api.com/)

## 🚀 Funcionalidades

- ✅ Conversão entre múltiplas moedas (USD, EUR, BRL, GBP, JPY, etc.)
- ✅ Taxas de câmbio atualizadas em tempo real
- ✅ Interface de linha de comando (CLI) intuitiva
- ✅ Histórico das últimas conversões realizadas
- ✅ Tratamento de erros e validações
- ✅ Código modular e bem estruturado
- ✅ Log gerado automático
